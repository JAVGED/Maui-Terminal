# Terminal - Aplikacja do zarządzania produktami (MAUI)

Prosta aplikacja w **.NET MAUI** do zarządzania listą produktów. Umożliwia:

- Dodawanie produktów (nazwa, cena, kategoria)  
- Usuwanie produktów po nazwie  
- Edytowanie danych produktów  
- Wyświetlanie listy produktów  
- Zamykanie aplikacji  

## Uruchomienie

1. Otwórz projekt w Visual Studio z obsługą .NET MAUI.  
2. Uruchom aplikację na wybranej platformie (Android, iOS, Windows).  
3. Korzystaj z przycisków w interfejsie do zarządzania produktami.
